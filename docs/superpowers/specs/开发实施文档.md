## 1. 开发环境配置

### 1.1 系统要求

- **操作系统：** Windows 10/11, macOS 12+, Ubuntu 20.04+
- **Python：** 3.13 (使用Anaconda)
- **Node.js：** 18+
- **包管理器：** pip, npm/pnpm

### 1.2 后端环境配置

```bash
# 进入后端目录
cd backend

# 激活Anaconda环境
conda activate base

# 安装依赖（使用已配置好的Python环境）
# 依赖已安装在用户Python目录
# 直接运行启动脚本

# 启动后端服务
# Windows: 双击 start.bat 或运行 start.ps1
start.bat

# 或手动启动
python -m uvicorn app.main:app --host 0.0.0.0 --port 8000
```

### 1.3 前端环境配置

```bash
# 进入前端目录
cd frontend

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 访问地址: http://localhost:5173
```

### 1.4 环境变量配置

**后端 .env 文件（已配置）：**

```env
# 应用配置
APP_NAME=知枢星图
APP_ENV=development
DEBUG=true
SECRET_KEY=your-secret-key-here-change-in-production

# 数据库配置
DATABASE_URL=sqlite:///./data/knowledge.db

# JWT 配置
JWT_SECRET_KEY=your-jwt-secret-key-here
JWT_ALGORITHM=HS256
JWT_EXPIRATION_DAYS=7

# AI 配置（通义千问 DashScope）
OPENAI_API_KEY=your-dashscope-api-key
OPENAI_BASE_URL=https://dashscope.aliyuncs.com/compatible-mode/v1
OPENAI_MODEL=qwen-turbo

# 向量嵌入配置（通义千问 text-embedding-v3）
DASHSCOPE_API_KEY=your-dashscope-api-key
DASHSCOPE_EMBEDDING_MODEL=text-embedding-v3

# Hugging Face 镜像配置（可选，如已使用 DashScope 可忽略）
HF_MIRROR_URL=https://hf-mirror.com
HF_CACHE_DIR=C:/temp/huggingface_cache

# 向量数据库配置（使用英文路径避免 FAISS 兼容问题）
FAISS_INDEX_PATH=C:/temp/zhishuxingtu_faiss
```

#### 安全配置说明

**API Key 安全措施：**

| 安全措施   | 说明                                          |
| ------ | ------------------------------------------- |
| 环境变量存储 | API Key 存储在后端 `.env` 文件中，通过 `os.environ` 读取 |
| 前后端分离  | 前端不包含任何 API Key，所有 AI 调用由后端完成               |
| Git 隔离 | `.env` 已加入 `.gitignore`，不会提交到仓库             |
| 日志保护   | API Key 不会出现在服务端日志中                         |

**本地开发 vs 生产部署：**

| 环境   | 配置方式                       |
| ---- | -------------------------- |
| 本地开发 | 使用 `.env` 文件，API Key 存储在本地 |
| 生产部署 | 建议使用系统环境变量，避免 `.env` 文件泄露  |

**生产环境配置示例：**

```powershell
# Windows 系统环境变量设置
setx OPENAI_API_KEY "your-api-key-here"
setx OPENAI_BASE_URL "https://dashscope.aliyuncs.com/compatible-mode/v1"
```

### 2.3 数据库模型实现（已完成）

#### users 表

```sql
CREATE TABLE users (
    id VARCHAR(36) PRIMARY KEY,
    username VARCHAR(100) NOT NULL UNIQUE,
    password_hash VARCHAR(255) NOT NULL,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);
```

#### folders 表

```sql
CREATE TABLE folders (
    id VARCHAR(36) PRIMARY KEY,
    name VARCHAR(200) NOT NULL,
    parent_id VARCHAR(36),
    level INTEGER CHECK (level IN (0, 1, 2, 3)),  -- 扩展至4层
    category_id VARCHAR(36),                       -- 新增：所属分类
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (parent_id) REFERENCES folders(id),
    FOREIGN KEY (category_id) REFERENCES categories(id)
);
```

#### notes 表

```sql
CREATE TABLE notes (
    id VARCHAR(36) PRIMARY KEY,
    title VARCHAR(500) NOT NULL,
    content TEXT,
    folder_id VARCHAR(36),
    user_id VARCHAR(36),
    linked_note_ids TEXT,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (folder_id) REFERENCES folders(id),
    FOREIGN KEY (user_id) REFERENCES users(id)
);
```

#### tags 表

```sql
CREATE TABLE tags (
    id VARCHAR(36) PRIMARY KEY,
    name VARCHAR(100) NOT NULL UNIQUE,
    color VARCHAR(7) DEFAULT '#00d4ff',
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);
```

#### note\_tags 表

```sql
CREATE TABLE note_tags (
    note_id VARCHAR(36),
    tag_id VARCHAR(36),
    PRIMARY KEY (note_id, tag_id),
    FOREIGN KEY (note_id) REFERENCES notes(id) ON DELETE CASCADE,
    FOREIGN KEY (tag_id) REFERENCES tags(id) ON DELETE CASCADE
);
```

**代码示例：**

```python
# app/services/llm_service.py
from langchain_openai import ChatOpenAI
from app.core.config import settings

class DashScopeLLM:
    _instance = None
    
    def __new__(cls):
        if cls._instance is None:
            cls._instance = super().__new__(cls)
            cls._instance._client = ChatOpenAI(
                model=settings.OPENAI_MODEL,
                api_key=settings.OPENAI_API_KEY,
                base_url=settings.OPENAI_BASE_URL,
                temperature=0.7
            )
        return cls._instance
    
    @property
    def client(self):
        return self._client
    
    def invoke(self, messages, **kwargs):
        return self._client.invoke(messages, **kwargs)

def get_llm_service() -> DashScopeLLM:
    return DashScopeLLM()
```

```python
# app/services/langchain_embeddings.py
from langchain_community.embeddings import DashScopeEmbeddings
from app.core.config import settings

class LangChainEmbeddings(DashScopeEmbeddings):
    def __init__(self):
        super().__init__(
            model=settings.DASHSCOPE_EMBEDDING_MODEL,
            dashscope_api_key=settings.DASHSCOPE_API_KEY
        )

_embedding_service = None

def get_langchain_embeddings() -> LangChainEmbeddings:
    global _embedding_service
    if _embedding_service is None:
        _embedding_service = LangChainEmbeddings()
    return _embedding_service
```

## 9. 后端数据模型更新

### 9.1 folders 表更新

**变更内容：** `level` 字段从 `CHECK (level IN (0, 1))` 改为 `CHECK (level IN (0, 1, 2, 3))`

```sql
CREATE TABLE folders (
    id VARCHAR(36) PRIMARY KEY,
    name VARCHAR(200) NOT NULL,
    parent_id VARCHAR(36),
    level INTEGER CHECK (level IN (0, 1, 2, 3)),  -- 扩展至4层
    category_id VARCHAR(36),                       -- 新增：所属分类
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (parent_id) REFERENCES folders(id),
    FOREIGN KEY (category_id) REFERENCES categories(id)
);
```

### 9.2 categories 表（内容分类）

```sql
CREATE TABLE categories (
    id VARCHAR(36) PRIMARY KEY,
    name VARCHAR(100) NOT NULL,
    icon VARCHAR(50),
    color VARCHAR(7) DEFAULT '#00d4ff',
    sort_order INTEGER DEFAULT 0,
    is_system BOOLEAN DEFAULT FALSE,  -- 系统预设分类不可删除
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);
```

**预置数据：**

| 分类ID     | 名称 | 图标 | 系统预设 |
| -------- | -- | -- | ---- |
| personal | 个人 | 👤 | 是    |
| work     | 工作 | 💼 | 是    |
| assets   | 素材 | 📦 | 是    |

### 9.3 content\_types 表（内容类型）

```sql
CREATE TABLE content_types (
    id VARCHAR(36) PRIMARY KEY,
    category_id VARCHAR(36) NOT NULL,
    name VARCHAR(100) NOT NULL,
    icon VARCHAR(50),
    color VARCHAR(7) DEFAULT '#00d4ff',
    field_schema TEXT,              -- JSON Schema，定义扩展字段
    is_system BOOLEAN DEFAULT FALSE,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (category_id) REFERENCES categories(id)
);
```

**预置数据：**

| 分类 | 内容类型 | 图标  |
| -- | ---- | --- |
| 个人 | 笔记   | 📝  |

| 素材 | 附件   | 📎  |
| 素材 | 链接收藏 | 🔗  |

### 9.4 contents 表（统一内容表）

```sql
CREATE TABLE contents (
    id VARCHAR(36) PRIMARY KEY,
    type_id VARCHAR(36) NOT NULL,          -- 内容类型ID
    category_id VARCHAR(36) NOT NULL,      -- 所属分类ID
    title VARCHAR(500) NOT NULL,
    content TEXT,
    extra_fields TEXT,                     -- JSON，存储扩展字段
    linked_content_ids TEXT,              -- JSON，关联的内容ID列表
    file_path VARCHAR(500),                -- 本地文件路径
    file_url VARCHAR(1000),               -- 外部链接
    file_size INTEGER,                     -- 文件大小
    folder_id VARCHAR(36),                 -- 所属文件夹
    user_id VARCHAR(36),
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (type_id) REFERENCES content_types(id),
    FOREIGN KEY (category_id) REFERENCES categories(id),
    FOREIGN KEY (folder_id) REFERENCES folders(id),
    FOREIGN KEY (user_id) REFERENCES users(id)
);
```

```python
class PromptChain:
    """提示词链：多步骤任务编排"""
    
    def __init__(self):
        self.steps = []
    
    def add_step(self, prompt_id: str, input_mapping: dict, output_key: str):
        """添加步骤"""
        self.steps.append({
            "prompt_id": prompt_id,
            "input_mapping": input_mapping,
            "output_key": output_key
        })
        return self
    
    def execute(self, initial_input: dict) -> dict:
        """执行链式调用"""
        context = initial_input.copy()
        
        for step in self.steps:
            # 1. 准备输入
            step_input = {}
            for var, source in step["input_mapping"].items():
                if source.startswith("$"):
                    step_input[var] = context.get(source[1:])
                else:
                    step_input[var] = source
            
            # 2. 调用AI
            result = self._call_ai(step["prompt_id"], step_input)
            
            # 3. 保存输出
            context[step["output_key"]] = result
        
        return context
```

```python
    def __init__(self):
        self.llm = get_llm_service()
        self.rag_chain = RAGChain()
        self.chat_chain = ChatChain()
    
    async def stream_rag_response(
        self,
        question: str,
        context: str,
        sources: List[Dict[str, Any]]
    ) -> AsyncGenerator[str, None]:
        """流式 RAG 问答响应"""
        try:
            # 构建提示词
            prompt = self._build_rag_prompt(question, context)
            
            # 流式调用 LLM
            async for chunk in self.llm.stream(prompt):
                data = json.dumps({
                    "type": "content",
                    "text": chunk
                }, ensure_ascii=False)
                yield f"data: {data}\n\n"
            
            # 发送来源信息
            sources_data = json.dumps({
                "type": "sources",
                "notes": sources
            }, ensure_ascii=False)
            yield f"data: {sources_data}\n\n"
            
            # 发送结束标记
            yield "data: [DONE]\n\n"
            
        except Exception as e:
            error_data = json.dumps({
                "type": "error",
                "message": str(e)
            }, ensure_ascii=False)
            yield f"data: {error_data}\n\n"
```

```typescript
export class SSEClient {
  private controller: AbortController | null = null;
  private reader: ReadableStreamDefaultReader<Uint8Array> | null = null;

  async connect(
    url: string,
    body: any,
    onMessage: (msg: SSEMessage) => void,
    onError?: (error: Error) => void
  ): Promise<void> {
    this.controller = new AbortController();
    
    try {
      const response = await fetch(url, {
        method: 'POST',
        headers: {
          'Content-Type': 'application/json',
          'Authorization': `Bearer ${localStorage.getItem('token')}`
        },
        body: JSON.stringify(body),
        signal: this.controller.signal
      });

      if (!response.ok) {
        throw new Error(`HTTP error! status: ${response.status}`);
      }

      this.reader = response.body?.getReader();
      if (!this.reader) {
        throw new Error('No reader available');
      }

      const decoder = new TextDecoder();
      let buffer = '';

      while (true) {
        const { done, value } = await this.reader.read();
        if (done) break;

        buffer += decoder.decode(value, { stream: true });
        const lines = buffer.split('\n');
        buffer = lines.pop() || '';

        for (const line of lines) {
          if (line.startsWith('data: ')) {
            const data = line.slice(6);
            if (data === '[DONE]') {
              onMessage({ type: 'done' });
              return;
            }
            try {
              const msg = JSON.parse(data) as SSEMessage;
              onMessage(msg);
            } catch (e) {
              console.error('Failed to parse SSE message:', data);
            }
          }
        }
      }
    } catch (error) {
      if ((error as Error).name !== 'AbortError') {
        onError?.(error as Error);
      }
    }
  }

  abort(): void {
    this.controller?.abort();
    this.reader?.cancel();
    this.controller = null;
    this.reader = null;
  }
}
```

```typescript
export class SSEClient {
  private maxRetries = 3;
  private retryDelay = 1000;

  async connectWithRetry(
    url: string,
    body: any,
    onMessage: (msg: SSEMessage) => void,
    onError?: (error: Error) => void
  ): Promise<void> {
    let retries = 0;
    
    while (retries < this.maxRetries) {
      try {
        await this.connect(url, body, onMessage, onError);
        return;
      } catch (error) {
        retries++;
        if (retries >= this.maxRetries) {
          onError?.(error as Error);
          throw error;
        }
        await new Promise(resolve => setTimeout(resolve, this.retryDelay * retries));
      }
    }
  }
}
```

### 18.1 问题背景

**当前分块策略问题**：

在 `embedding_service.py` 中，当前使用固定长度截断：

```python
def prepare_note_text(self, title: str, content: str, max_length: int = 2048) -> str:
    combined = f"{title}\n{content or ''}"
    if len(combined) > max_length:
        combined = combined[:max_length]  # ❌ 固定长度截断
    return combined
```

**这会导致严重问题**：

| 问题 | 举例 |
|------|------|
| 信息丢失 | 3000字符的笔记被截断到2048，后952字符内容完全丢失 |
| 语义割裂 | "Python教程"第10章的重要内容可能被腰斩 |
| 上下文丢失 | 列表项、设计方案等完整内容被截断 |

### 18.2 优化方案

**统一递归分块 + 50% Overlap**：

```
原文笔记（AI精炼后）：

内容...

---
# 实现方法

---

块3: # 最佳实践\n内容...（按段落递归切割后保留完整 ✅）

50% Overlap：
块1: [内容1] + [内容2后400字]
块2: [内容2前400字] + [内容3]
```

| Level | 分割方式 | 说明 |
|-------|---------|------|
| 1 | `---` | AI分块标记，最优先 |
| 2 | Markdown 标题 | `#` 到 `######` |

```python
import re
from typing import List, Dict, Any, Tuple
from dataclasses import dataclass

@dataclass
class TextChunk:
    content: str
    chunk_index: int
    total_chunks: int
    note_id: str
    metadata: Dict[str, Any]

class RecursiveChunker:
    def __init__(
        self,
        max_chunk_size: int = 800,
        overlap_ratio: float = 0.5,
        min_chunk_size: int = 50
    ):
        self.max_chunk_size = max_chunk_size
        self.overlap_ratio = overlap_ratio
        self.min_chunk_size = min_chunk_size

    def split_by_divider(self, text: str) -> List[str]:
        """按 --- 分块标记分割"""
        parts = text.split('\n---\n')
        return [p.strip() for p in parts if p.strip()]

    def split_by_headings(self, text: str) -> List[str]:
        """按 Markdown 标题分割"""
        heading_pattern = r'^(#{1,6}\s+.+)$'
        lines = text.split('\n')

        sections = []
        current_section = []

        for line in lines:
            if re.match(heading_pattern, line, re.MULTILINE):
                if current_section:
                    sections.append('\n'.join(current_section))
                    current_section = []
            current_section.append(line)

        if current_section:
            sections.append('\n'.join(current_section))

        return [s.strip() for s in sections if s.strip()]

    def split_by_paragraphs(self, text: str) -> List[str]:
        """按段落分割"""
        paragraphs = re.split(r'\n\n+', text)
        return [p.strip() for p in paragraphs if p.strip()]

    def split_by_sentences(self, text: str) -> List[str]:
        """按句子分割（保底策略）"""
        sentence_pattern = r'(?<=[。！？.!?])\s*'
        sentences = re.split(sentence_pattern, text)
        return [s.strip() for s in sentences if s.strip()]

    def add_overlap(self, chunks: List[str]) -> List[Tuple[str, str]]:
        """添加50%重叠，返回(chunk, overlap_start)元组列表"""
        if len(chunks) <= 1:
            return [(c, "") for c in chunks]

        overlap_size = int(self.max_chunk_size * self.overlap_ratio)
        result = []

        for i, chunk in enumerate(chunks):
            if len(chunk) <= self.max_chunk_size:
                if i < len(chunks) - 1:
                    next_overlap = chunks[i + 1][:overlap_size]
                    combined = chunk + "\n" + next_overlap
                else:
                    combined = chunk
                result.append((combined, chunk[:50] if len(chunk) > 50 else chunk))
            else:
                result.append((chunk[:self.max_chunk_size], chunk[:50]))

        return result

    def recursive_chunk(self, text: str, level: int = 1) -> List[str]:
        """递归分块主方法"""
        if not text or not text.strip():
            return []

        if len(text) <= self.max_chunk_size:
            return [text]

        chunks = []

        if level == 1:
            chunks = self.split_by_divider(text)
        elif level == 2:
            chunks = self.split_by_headings(text)
        elif level == 3:
            chunks = self.split_by_paragraphs(text)
        elif level == 4:
            chunks = self.split_by_sentences(text)

        if len(chunks) == 1 or all(len(c) <= self.max_chunk_size for c in chunks):
            return chunks

        if level < 4:
            merged = []
            for chunk in chunks:
                if len(chunk) <= self.max_chunk_size:
                    merged.append(chunk)
                else:
                    sub_chunks = self.recursive_chunk(chunk, level + 1)
                    merged.extend(sub_chunks)
            return merged

        return [text[:self.max_chunk_size]]

    def chunk(self, note_id: str, title: str, content: str) -> List[TextChunk]:
        """主分块方法"""
        if not content or not content.strip():
            return [TextChunk(
                content=f"{title}",
                chunk_index=0,
                total_chunks=1,
                note_id=note_id,
                metadata={"type": "title_only"}
            )]

        combined = f"{title}\n{content}"
        raw_chunks = self.recursive_chunk(combined, level=1)

        chunks_with_overlap = self.add_overlap(raw_chunks)

        result_chunks = []
        for i, (chunk_content, _) in enumerate(chunks_with_overlap):
            if len(chunk_content) >= self.min_chunk_size:
                result_chunks.append(TextChunk(
                    content=chunk_content,
                    chunk_index=i,
                    total_chunks=len(chunks_with_overlap),
                    note_id=note_id,
                    metadata={"level": 1}
                ))

        total = len(result_chunks)
        for c in result_chunks:
            c.total_chunks = total

        return result_chunks
```

```python
            self.add_vector(chunk_id, vector)
            chunk_ids.append(chunk_id)
        
        self._note_chunks_map[note_id] = chunk_ids
        return {"success": len(chunk_ids)}
    
    def search(self, query_vector: np.ndarray, k: int = 5, threshold: float = 0.3) -> List:
        """搜索时按笔记ID聚合结果"""
        raw_results = self._original_store.search(query_vector, k * 3, threshold)
        
        note_scores: Dict[str, List[float]] = {}
        for chunk_id, score in raw_results:
            if "_chunk_" in chunk_id:
                note_id = chunk_id.split("_chunk_")[0]
                if note_id not in note_scores:
                    note_scores[note_id] = []
                note_scores[note_id].append(score)
        
        aggregated = [(note_id, max(scores)) for note_id, scores in note_scores.items()]
        aggregated.sort(key=lambda x: x[1], reverse=True)
        
        return aggregated[:k]
```

```python
from rank_bm25 import BM25Okapi
import jieba
from typing import List, Dict, Tuple

class BM25Service:
    def __init__(self):
        self.tokenized_corpus: List[List[str]] = []
        self.bm25 = None
        self.documents: List[Dict] = []
    
    def index_documents(self, docs: List[Dict]) -> Dict:
        """构建 BM25 索引"""
        self.documents = docs
        self.tokenized_corpus = [jieba.lcut(doc['content']) for doc in docs]
        self.bm25 = BM25Okapi(self.tokenized_corpus)
        return {"indexed": len(docs)}
    
    def add_document(self, doc: Dict) -> None:
        """添加单个文档到索引"""
        self.documents.append(doc)
        tokens = jieba.lcut(doc['content'])
        self.tokenized_corpus.append(tokens)
        self.bm25 = BM25Okapi(self.tokenized_corpus)
    
    def search(self, query: str, top_k: int = 15) -> List[Tuple[str, float]]:
        """BM25 检索"""
        if not self.bm25:
            return []
        
        query_tokens = jieba.lcut(query)
        scores = self.bm25.get_scores(query_tokens)
        
        doc_scores = []
        for i, score in enumerate(scores):
            if i < len(self.documents):
                doc_scores.append((self.documents[i]['id'], score))
        
        doc_scores.sort(key=lambda x: x[1], reverse=True)
        return doc_scores[:top_k]
    
    def clear(self) -> None:
        """清空索引"""
        self.tokenized_corpus = []
        self.bm25 = None
        self.documents = []
```

```python
from typing import List, Tuple, Dict
from app.services.bm25_service import get_bm25_service
from app.services.vector_store_adapter import get_vector_store_adapter

def reciprocal_rank_fusion(
    vector_results: List[Tuple[str, float]],
    bm25_results: List[Tuple[str, float]],
    k: int = 60,
    vector_weight: float = 0.6,
    bm25_weight: float = 0.4
) -> List[Tuple[str, float]]:
    """RRF 混合融合"""
    rrf_scores: Dict[str, float] = {}
    
    # 向量检索结果
    for rank, (doc_id, score) in enumerate(vector_results):
        rrf_scores[doc_id] = rrf_scores.get(doc_id, 0) + vector_weight * (1 / (k + rank + 1))
    
    # BM25 结果
    for rank, (doc_id, score) in enumerate(bm25_results):
        rrf_scores[doc_id] = rrf_scores.get(doc_id, 0) + bm25_weight * (1 / (k + rank + 1))
    
    # 排序
    sorted_results = sorted(rrf_scores.items(), key=lambda x: x[1], reverse=True)
    return sorted_results

class HybridSearchService:
    def __init__(self):
        self.bm25_service = get_bm25_service()
        self.vector_store = get_vector_store_adapter()
        self.vector_weight = 0.6
        self.bm25_weight = 0.4
        self.rrf_k = 60
        self.top_k = 10
    
    def search(self, query: str, top_k: int = None) -> List[Tuple[str, float]]:
        """混合检索"""
        if top_k is None:
            top_k = self.top_k
        
        # 向量检索
        query_vector = self.vector_store.embed_query(query)
        vector_results = self.vector_store.search(query_vector, top_k * 2)
        
        # BM25 检索
        bm25_results = self.bm25_service.search(query, top_k * 2)
        
        # RRF 融合
        fused_results = reciprocal_rank_fusion(
            vector_results,
            bm25_results,
            k=self.rrf_k,
            vector_weight=self.vector_weight,
            bm25_weight=self.bm25_weight
        )
        
        return fused_results[:top_k]
    
    def rebuild_bm25_index(self, docs: List[Dict]) -> Dict:
        """重建 BM25 索引"""
        return self.bm25_service.index_documents(docs)
```

```python
from ragas import evaluate
from ragas.metrics import (
    answer_relevance,
    context_relevance,
    faithfulness,
    answer_correctness
)
from datasets import Dataset
from typing import List, Dict
import json

class RAGASEvaluator:
    def __init__(self):
        self.metrics = [
            answer_relevance,
            context_relevance,
            faithfulness
        ]
        self.metrics_with_ground_truth = [
            answer_relevance,
            context_relevance,
            faithfulness,
            answer_correctness
        ]
    
    def evaluate(self, test_cases: List[Dict], use_ground_truth: bool = False) -> Dict:
        """执行 RAGAS 评估"""
        metrics = self.metrics_with_ground_truth if use_ground_truth else self.metrics
        
        # 准备数据集
        data = {
            "question": [tc['question'] for tc in test_cases],
            "answer": [tc.get('answer', '') for tc in test_cases],
            "contexts": [tc.get('contexts', []) for tc in test_cases],
        }
        
        if use_ground_truth:
            data["ground_truth"] = [tc.get('ground_truth', '') for tc in test_cases]
        
        dataset = Dataset.from_dict(data)
        
        # 执行评估
        result = evaluate(dataset, metrics=metrics)
        
        return {
            "answer_relevance": result["answer_relevance"] if hasattr(result, 'answer_relevance') else None,
            "context_relevance": result["context_relevance"] if hasattr(result, 'context_relevance') else None,
            "faithfulness": result["faithfulness"] if hasattr(result, 'faithfulness') else None,
            "answer_correctness": result["answer_correctness"] if hasattr(result, 'answer_correctness') else None
        }
    
    def evaluate_single(self, question: str, answer: str, contexts: List[str], ground_truth: str = None) -> Dict:
        """评估单个案例"""
        test_cases = [{
            "question": question,
            "answer": answer,
            "contexts": contexts,
            "ground_truth": ground_truth or ""
        }]
        return self.evaluate(test_cases, use_ground_truth=bool(ground_truth))

# 单例
```

```python
import json
from typing import List, Dict
from pathlib import Path

class EvaluationDataset:
    def __init__(self, storage_path: str = "data/evaluation"):
        self.storage_path = Path(storage_path)
        self.storage_path.mkdir(parents=True, exist_ok=True)
        self._datasets: Dict[str, List[Dict]] = {}
    
    def load_dataset(self, name: str) -> List[Dict]:
        """加载评估数据集"""
        file_path = self.storage_path / f"{name}.json"
        if file_path.exists():
            with open(file_path, 'r', encoding='utf-8') as f:
                data = json.load(f)
                self._datasets[name] = data.get('test_cases', [])
                return self._datasets[name]
        return []
    
    def save_dataset(self, name: str, dataset: Dict) -> None:
        """保存评估数据集"""
        file_path = self.storage_path / f"{name}.json"
        with open(file_path, 'w', encoding='utf-8') as f:
            json.dump(dataset, f, ensure_ascii=False, indent=2)
        self._datasets[name] = dataset.get('test_cases', [])
    
    def list_datasets(self) -> List[str]:
        """列出所有评估数据集"""
        return [f.stem for f in self.storage_path.glob("*.json")]
    
    def delete_dataset(self, name: str) -> bool:
        """删除评估数据集"""
        file_path = self.storage_path / f"{name}.json"
        if file_path.exists():
            file_path.unlink()
            if name in self._datasets:
                del self._datasets[name]
            return True
        return False
    
    def get_dataset(self, name: str) -> List[Dict]:
        """获取数据集（从缓存或文件）"""
        if name not in self._datasets:
            return self.load_dataset(name)
        return self._datasets[name]
```

```python
import numpy as np
from typing import List, Dict
from datetime import datetime

class EvaluationReport:
    def generate_report(self, eval_results: List[Dict]) -> Dict:
        """生成评估报告"""
        metrics = ['answer_relevance', 'context_relevance', 'faithfulness', 'answer_correctness']
        
        report = {
            "generated_at": datetime.now().isoformat(),
            "total_cases": len(eval_results),
            "summary": {},
            "distribution": {},
            "low_score_cases": [],
            "recommendations": []
        }
        
        # 计算各指标统计
        for metric in metrics:
            scores = [r[metric] for r in eval_results if metric in r and r[metric] is not None]
            if scores:
                report["summary"][metric] = {
                    "mean": float(np.mean(scores)),
                    "std": float(np.std(scores)),
                    "min": float(np.min(scores)),
                    "max": float(np.max(scores)),
                    "median": float(np.median(scores))
                }
                report["distribution"][metric] = self._calculate_distribution(scores)
        
        # 低分案例（忠实度 < 0.5）
        for i, r in enumerate(eval_results):
            if r.get('faithfulness', 1) < 0.5:
                report["low_score_cases"].append({
                    "case_index": i,
                    "question": r.get('question', ''),
                    "faithfulness": r.get('faithfulness', 0)
                })
        
        # 优化建议
        report["recommendations"] = self._generate_recommendations(report["summary"])
        
        return report
    
    def _calculate_distribution(self, scores: List[float], bins: int = 10) -> Dict:
        """计算分数分布"""
        hist, edges = np.histogram(scores, bins=bins, range=(0, 1))
        return {
            "histogram": [int(h) for h in hist],
            "edges": [float(e) for e in edges]
        }
    
    def _generate_recommendations(self, summary: Dict) -> List[str]:
        """生成优化建议"""
        recommendations = []
        mapping = {
            "answer_relevance": "提示词模板",
            "context_relevance": "检索策略和分块方法",
            "faithfulness": "上下文质量和提示词忠实度",
            "answer_correctness": "知识库内容准确性"
        }
        
        for metric, stats in summary.items():
            if stats["mean"] < 0.6:
                recommendations.append(
                    f"{metric} 较低（{stats['mean']:.2f}），建议优化{mapping.get(metric, '相关模块')}。"
                )
        
        if not recommendations:
            recommendations.append("各项指标表现良好，继续保持！")
        
        return recommendations
```

```python
from transformers import CLIPProcessor, CLIPModel
from PIL import Image
import torch
import numpy as np
from typing import List

class CLIPService:
    def __init__(self, model_name: str = "openai/clip-vit-base-patch32"):
        self.model_name = model_name
        self.model = None
        self.processor = None
        self.device = "cuda" if torch.cuda.is_available() else "cpu"
    
    def load_model(self):
        """加载 CLIP 模型"""
        if self.model is None:
            self.model = CLIPModel.from_pretrained(self.model_name)
            self.processor = CLIPProcessor.from_pretrained(self.model_name)
            self.model.to(self.device)
            self.model.eval()
    
    def encode_image(self, image_path: str) -> np.ndarray:
        """图片向量化"""
        self.load_model()
        
        image = Image.open(image_path).convert("RGB")
        inputs = self.processor(images=image, return_tensors="pt").to(self.device)
        
        with torch.no_grad():
            image_features = self.model.get_image_features(**inputs)
        
        # 归一化
        features = image_features.cpu().numpy()[0]
        features = features / np.linalg.norm(features)
        
        return features
    
    def encode_text(self, text: str) -> np.ndarray:
        """文本向量化"""
        self.load_model()
        
        inputs = self.processor(text=[text], return_tensors="pt", padding=True).to(self.device)
        
        with torch.no_grad():
            text_features = self.model.get_text_features(**inputs)
        
        # 归一化
        features = text_features.cpu().numpy()[0]
        features = features / np.linalg.norm(features)
        
        return features
    
    def similarity(self, image_path: str, text: str) -> float:
        """计算图片-文本相似度"""
        image_features = self.encode_image(image_path)
        text_features = self.encode_text(text)
        
        return float(np.dot(image_features, text_features))
    
    def encode_images_batch(self, image_paths: List[str]) -> np.ndarray:
        """批量图片向量化"""
        self.load_model()
        
        images = [Image.open(path).convert("RGB") for path in image_paths]
        inputs = self.processor(images=images, return_tensors="pt", padding=True).to(self.device)
        
        with torch.no_grad():
            image_features = self.model.get_image_features(**inputs)
        
        # 归一化
        features = image_features.cpu().numpy()
        norms = np.linalg.norm(features, axis=1, keepdims=True)
        features = features / norms
        
        return features

# 单例
_clip_service = None

def get_clip_service() -> CLIPService:
    global _clip_service
    if _clip_service is None:
        _clip_service = CLIPService()
    return _clip_service
```

```python
import faiss
import numpy as np
from typing import List, Dict, Tuple
from app.services.clip_service import get_clip_service

class MultimodalIndex:
    def __init__(self):
        self.clip_service = get_clip_service()
        self.image_vectors: Dict[str, np.ndarray] = {}
        self.image_metadata: Dict[str, Dict] = {}
        self.faiss_index = None
        self.image_ids: List[str] = []
        self.dimension = 512
    
    def add_image(self, image_id: str, image_path: str, metadata: Dict) -> None:
        """添加图片到索引"""
        vector = self.clip_service.encode_image(image_path)
        self.image_vectors[image_id] = vector
        self.image_metadata[image_id] = metadata
    
    def build_index(self) -> None:
        """构建 FAISS 索引"""
        if not self.image_vectors:
            return
        
        # 使用 Inner Product（余弦相似度，因为向量已归一化）
        self.faiss_index = faiss.IndexFlatIP(self.dimension)
        self.image_ids = list(self.image_vectors.keys())
        
        vectors = np.array([
            self.image_vectors[img_id] for img_id in self.image_ids
        ], dtype=np.float32)
        
        self.faiss_index.add(vectors)
    
    def search_by_text(self, query: str, top_k: int = 5) -> List[Dict]:
        """文本搜索图片"""
        if not self.faiss_index:
            return []
        
        query_vector = self.clip_service.encode_text(query)
        query_vector = query_vector.reshape(1, -1).astype(np.float32)
        
        scores, indices = self.faiss_index.search(query_vector, top_k)
        
        results = []
        for score, idx in zip(scores[0], indices[0]):
            if 0 <= idx < len(self.image_ids):
                img_id = self.image_ids[idx]
                results.append({
                    "image_id": img_id,
                    "score": float(score),
                    "metadata": self.image_metadata[img_id]
                })
        
        return results
    
    def search_by_image(self, query_image_path: str, top_k: int = 5) -> List[Dict]:
        """图片搜索图片"""
        if not self.faiss_index:
            return []
        
        query_vector = self.clip_service.encode_image(query_image_path)
        query_vector = query_vector.reshape(1, -1).astype(np.float32)
        
        scores, indices = self.faiss_index.search(query_vector, top_k)
        
        results = []
        for score, idx in zip(scores[0], indices[0]):
            if 0 <= idx < len(self.image_ids):
                img_id = self.image_ids[idx]
                results.append({
                    "image_id": img_id,
                    "score": float(score),
                    "metadata": self.image_metadata[img_id]
                })
        
        return results
    
    def remove_image(self, image_id: str) -> bool:
        """从索引删除图片"""
        if image_id in self.image_vectors:
            del self.image_vectors[image_id]
            del self.image_metadata[image_id]
            self.build_index()  # 重建索引
            return True
        return False
    
    def clear(self) -> None:
        """清空索引"""
        self.image_vectors = {}
        self.image_metadata = {}
        self.faiss_index = None
        self.image_ids = []
    
    def save(self, path: str) -> None:
        """保存索引到磁盘"""
        if self.faiss_index is not None:
            faiss.write_index(self.faiss_index, f"{path}.index")
        
        import pickle
        with open(f"{path}_metadata.pkl", 'wb') as f:
            pickle.dump({
                'image_ids': self.image_ids,
                'image_metadata': self.image_metadata
            }, f)
    
    def load(self, path: str) -> None:
        """从磁盘加载索引"""
        try:
            self.faiss_index = faiss.read_index(f"{path}.index")
            
            import pickle
            with open(f"{path}_metadata.pkl", 'rb') as f:
                data = pickle.load(f)
                self.image_ids = data['image_metadata']
                self.image_metadata = data['image_metadata']
        except Exception:
            pass

# 单例
_multimodal_index = None

def get_multimodal_index() -> MultimodalIndex:
    global _multimodal_index
    if _multimodal_index is None:
        _multimodal_index = MultimodalIndex()
    return _multimodal_index
```

```python
from app.services.clip_service import get_clip_service
from app.services.llm_service import get_llm_service
from PIL import Image
from typing import List

class MultimodalQA:
    def __init__(self):
        self.clip_service = get_clip_service()
        self.llm_service = get_llm_service()
    
    async def answer_about_image(self, question: str, image_path: str) -> str:
        """基于图片内容回答问题"""
        image_description = self._describe_image(image_path)
        
        prompt = f"""根据图片内容回答问题。

图片描述：{image_description}

问题：{question}

请基于图片内容回答问题。"""
        
        response = await self.llm_service.invoke([
            {"role": "user", "content": prompt}
        ])
        
        return response.content if hasattr(response, 'content') else str(response)
    
    def _describe_image(self, image_path: str) -> str:
        """使用 CLIP 分析图片"""
        candidates = [
            "a photo containing a cat",
            "a photo containing a dog",
            "a photo containing a landscape",
            "a photo containing a person",
            "a photo containing text or code",
            "a photo containing food",
            "a photo containing a building",
            "a photo containing a car",
            "a photo containing nature",
            "a photo containing an animal"
        ]
        
        image = Image.open(image_path).convert("RGB")
        inputs = self.clip_service.processor(
            text=candidates,
            images=image,
            return_tensors="pt",
            padding=True
        )
        
        with __import__('torch').no_grad():
            outputs = self.clip_service.model(**inputs)
        
        logits_per_image = outputs.logits_per_image
        probs = logits_per_image.softmax(dim=1)
        
        top_idx = probs.argmax().item()
        return f"这张图片最可能是{candidates[top_idx].replace('a photo containing a ', '')}"
```

```python
from fastapi import APIRouter, UploadFile, File, Form
from app.services.multimodal_index import get_multimodal_index
from app.services.multimodal_qa import get_multimodal_qa
import os

router = APIRouter(prefix="/multimodal", tags=["多模态"])

@router.post("/index")
async def add_image_to_index(
    file: UploadFile = File(...),
    note_id: str = Form(...),
    metadata: str = Form("{}")
):
    """添加图片到索引"""
    import json
    import uuid
    
    # 保存图片
    image_dir = "data/multimodal"
    os.makedirs(image_dir, exist_ok=True)
    
    image_id = str(uuid.uuid4())
    image_path = f"{image_dir}/{image_id}.jpg"
    
    with open(image_path, "wb") as f:
        content = await file.read()
        f.write(content)
    
    # 添加到索引
    multimodal_index = get_multimodal_index()
    multimodal_index.add_image(
        image_id=image_id,
        image_path=image_path,
        metadata={
            "note_id": note_id,
            "path": image_path,
            "extra": json.loads(metadata)
        }
    )
    multimodal_index.build_index()
    
    return {"success": True, "image_id": image_id}

@router.post("/search")
async def search_images(
    query: str = Form(...),
    top_k: int = Form(5)
):
    """跨模态检索"""
    multimodal_index = get_multimodal_index()
    results = multimodal_index.search_by_text(query, top_k)
    
    return {"results": results}

@router.post("/qa")
async def image_qa(
    question: str = Form(...),
    image_id: str = Form(...)

    multimodal_index = get_multimodal_index()
    image_path = multimodal_index.image_metadata.get(image_id, {}).get("path")
    
    if not image_path:
        return {"error": "Image not found"}
    
    multimodal_qa = get_multimodal_qa()
    answer = await multimodal_qa.answer_about_image(question, image_path)
    
    return {"answer": answer}

@router.get("/status")

    """获取多模态索引状态"""
    multimodal_index = get_multimodal_index()
    return {
        "indexed_images": len(multimodal_index.image_ids),
        "is_ready": multimodal_index.faiss_index is not None
    }
```
